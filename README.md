# MyPlayer

Small music player for windows, mac and linux

![Scriptshit MyPlayer](http://pub.scriptshit.de//MyPlayer-20151023-010523.jpg)

* automatic playlist generation. current dir and all sub-dirs
* wave display
* id3 support
* win, mac, linux
